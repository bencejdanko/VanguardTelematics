# PCF8591 ADC / DAC

This chip has 4 ADC (analog-to-digital channels) and one DAC (digital to analog channel).

## Content

### pc8591-10kpot

This folder contains code for a simple demonstration of how to use a PCF8591 chip
with the RaspBerry PI 4B to read the resistance of a 10K potentiometer, commonly
used for adjustments on smaller boards (usually with trim pots) or larger potentiometers
with knobs are used to adjust settings or volume.

[pc8591-10kpot](pcf8591-10kpot)

### pcf8591-10kpot-led-control

This folder contains a sample application demonstrates how to use a PCF8591
chip with the RaspBerry PI 4B to read the resistance of a 10K potentiometer,
and feed that value to the PCF8591 analog output to control the brightness of an LED.

[pcf8591-10kpot-led-control](pcf8591-10kpot-led-control)

### pcf8591-ggboard

This folder contains a sample application demonstrates how to use a PCF8591 chip with the
RaspBerry PI 4B to read the values from the different devices incorporated into the
Gump's Grocer PC8951 board.

[pcf8591-ggboard](pcf8591-ggboard)
